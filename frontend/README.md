# Support Haven Frontend

React application for Support Haven platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.

## Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- React Router v6
- TailwindCSS
- Vite
- Axios
- Socket.io Client
